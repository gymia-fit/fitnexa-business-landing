# fitnexa-landing

Gymia marketing landing page. Next.js 15 + Tailwind 4 + Dark theme.

## Environments

| Environment | Branch | URL |
|-------------|--------|-----|
| Production | master | [gymia.fit](https://gymia.fit) |
| UAT | develop | [uat.gymia.fit](https://uat.gymia.fit) |

## Setup

```bash
cp .env.example .env.local
npm install
npm run dev
```

## Deploy

Deploy to Vercel. Connect the GitHub repo and set environment variables.
